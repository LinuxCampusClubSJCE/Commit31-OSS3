# Commit31-OSS3 (UniFind)

## 📌 Overview
UniFind is an open-source Lost & Found platform designed for college campuses.  
It helps students report lost items, post found belongings, and securely reconnect owners through smart matching and verification workflows.

---

## ✨ Features
- 📦 Post lost items
- 🔎 Post found items
- 🔐 Secure ownership claim system
- 🎓 Campus verification login
- 💬 Real-time communication

---

## 🛠 Tech Stack
- Frontend: React
- Backend: FastAPI
- Database: PostgreSQL
- Styling: TailwindCSS
